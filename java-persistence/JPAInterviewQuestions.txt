# Top 40 JPA + Spring Data JPA Interview Questions

## Tier-1: Core Must-Know (Very Frequently Asked)

These are asked in **almost every interview**.

1. What is **JPA**, and how is it different from **Hibernate**?

2. What is an **Entity** in JPA? What annotations are required to define one?

3. Explain **EntityManager** and its role in JPA.

4. What is a **Persistence Context**?

5. What is **dirty checking** in JPA?

6. What is the difference between
   **persist()** and **merge()**?

7. What is **JPQL**, and how is it different from SQL?

8. What are the different **types of relationships** in JPA?

* One-To-One
* One-To-Many
* Many-To-One
* Many-To-Many

9. What is **mappedBy**, and how is it different from **@JoinColumn**?

   Explain Composite Primary key in JPA

10. What is the difference between
    **LAZY** and **EAGER** fetching?

11. What is the **N+1 select problem**, and how do you fix it?
    Using JOIN FETCH or EntityGraph(attributePaths = {...})
12. What is **CascadeType**? Explain common cascade types.

13. What is **orphanRemoval**, and how is it different from cascade delete?
  orphanRemoval is necessary when child needs to be deleted when relationShip between parent and child is broken. Whereas CascaseType.REMOVE removes children only during parent deletion
14. What are **bidirectional** and **unidirectional** relationships?

15. How do you handle **infinite recursion** in bidirectional mapping?

(Json annotations or DTO usage)

16. What is **Spring Data JPA**, and how does it simplify JPA usage?

17. What is **JpaRepository**?

18. What are the different **Spring Data repository types**?

* CrudRepository
* PagingAndSortingRepository
* JpaRepository

19. How does **Spring Data JPA handle transactions by default**?

20. What is **@Transactional**, and how does it work?

21. What is the difference between
    **@Query** and **query derivation methods**?

22. How do you write a **native SQL query** in Spring Data JPA?

23. How do you implement **pagination and sorting**?

24. What is **EntityGraph**, and when do you use it?

25. What is **Specification API**, and when is it used?

26. How do you build **dynamic queries**?

33. How does **JPA inheritance** work?

Strategies:

35. What is **pessimistic locking**?
